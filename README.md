# 증강체스 공개방 확장프로그램 (비공식)

[augmentchess.org](https://augmentchess.org/)에 **공개방 · 채팅 · 쉬운 재입장** 기능을 더해주는 비공식 커뮤니티 서드파티 확장프로그램의 다운로드 페이지입니다.

> ⚠️ 이 프로젝트는 증강체스 공식 기능이 아닌 **비공식 커뮤니티 도구**입니다.
> 향후 공식 업데이트로 일부 기능이 중복되거나 동작하지 않을 수 있습니다.

## 호스팅

이 저장소는 단일 정적 페이지(`index.html`)이며 GitHub Pages로 서빙됩니다.

- 사이트 주소: `https://uki0903.github.io/Arguchess-thirdparty-web/`
- 활성화: 저장소 **Settings → Pages → Source: `Deploy from a branch` → `main` / `/ (root)`**
